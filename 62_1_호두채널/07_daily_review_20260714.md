## #1 — The Desk 베타 오픈 및 Surge Desk 종목 분석 기능 소개
🔗 https://youtu.be/RJnEMixTJlw | 📅 2026-07-13 | 👁 578 | 📂 자동매매 시스템 구축

### 📋 핵심 요약
호두 채널 자체 서비스 'The Desk(SurgeDesk)'의 베타 오픈 안내 영상. 투자 대가 스크린 기반 종목 추천, 재무/차트/수급/정책 분석, AI 유망주 리스트, 매일 기관 리포트를 요약해주는 "모닝 라디오" 기능까지 포함된 웹 서비스. 신규 가입 시 3일 무료.

### 🔧 주요 기술/도구
- 종목 추천 스크린 (국장 용법, 투자 대가 기준)
- 차트 분석: 이동평균선, RS 라인, 수급, 정책 공시 반영
- 딥다이브 분석 (반도체·로봇·정책 산업 등 메가 트렌드)
- 모닝 라디오 — 기관 리포트 AI 요약 오디오 (일 30여 개 소스)

### 💡 핵심 학습 내용
- 지금까지 유튜브 채널에서 소개만 하던 파이프라인 결과물이 실제 웹 서비스(SurgeDesk)로 나왔다는 게 이번 영상의 핵심
- 3일 무료 체험이 있어 우리 시스템(종가베팅/AlphaForge)과 결과를 직접 비교해볼 수 있는 벤치마크로 쓸 수 있음
- 이 채널은 "유사투자자문업" 신고 사업자 — 법적으로 개인 맞춤 자문은 아니고 정보 제공용이라는 점 명시됨 (참고용 데이터 소스로만 취급할 것)

### 🎯 바로보기님 적용 포인트
목표: n8n + AI로 주식 계좌 자동화 (뿌리 ⑤자동화 / ⑦재테크)
- 우리가 만들고 있는 종가베팅/AlphaForge 파이프라인의 "완성형 최종 모습"을 미리 보여주는 참고 사례. 딱히 따라 만들 필요는 없고, 3일 무료 기간에 SurgeDesk 결과 vs 우리 시스템 결과를 대조해서 우리 로직의 허점을 찾는 용도로 쓰면 좋음.

## #2 — 13명의 투자 대가(워런 버핏, 벤저민 그레이엄, 피터 린치 등)의 투자 기준을 노드(Node) 분석
🔗 https://youtu.be/QC3yJ-blDAE | 📅 2026-07-13 | 👁 253 | 📂 자동매매 시스템 구축 — ⭐우선순위 높음

### 📋 핵심 요약
13명의 투자 대가 기준을 노드 기반 워크플로우로 구현해 KRX 전체 종목을 필터링하는 시스템. 거래대금 필터(120개 추출) → 대가별 기준 교집합(증거 합의) → AI 브레인 점수 + 수급 분석 → 최종 유망 종목(신한지주, LG이노텍, 대한항공, 현대로템 등) 산출까지 전 과정 시연.

### 🔧 주요 기술/도구
- KRX 데이터 기반 1차 필터링 (거래대금·실매매가능성)
- Evidence Consensus(증거 합의) — 대가 기준 2개 이상 동시 충족 종목만 추림
- AI Brain 점수 + 외국인/기관 수급 결합 스코어링
- 위험요소 배제(평균 이하 거래량, 하락 알림, 공매도 비중)로 최종 순위화
- Notion 프롬프트 자료 공개 (노드 구성 그대로 재현 가능)

### 💡 핵심 학습 내용
- "여러 기준의 교집합만 통과"라는 필터링 방식이 우리 종가베팅 S/A/B 등급 로직보다 한 단계 더 엄격함 — 오탐(false positive) 줄이는 데 참고할 만함
- Notion에 프롬프트 원본이 공개돼 있어 노드 구조를 그대로 참고 가능 (강의 아니어도 무료 자료로 충분히 재현 가능한 수준)

### 🎯 바로보기님 적용 포인트
목표: n8n + AI로 주식 계좌 자동화 (뿌리 ⑤자동화 / ⑦재테크) — ⭐이번 3개 중 가장 직접 적용 가치 높음
- 지금 종가베팅 시스템의 "등급 산정 로직"에 이 영상의 Evidence Consensus 개념(여러 대가 기준 동시 충족)을 얹으면, S등급 종목의 오탐률을 낮출 수 있음
- Notion 프롬프트가 공개돼 있으니 그대로 가져와서 우리 파이프라인의 필터링 노드 하나로 추가하는 실험 해볼 만함 (구현 난이도 낮음 — 프롬프트 복붙 수준)

## #3 — ⭐9만 스타로 AI 금융 프로젝트 부동의 1위 오픈소스입니다 (TradingAgents)
🔗 https://youtu.be/82UVtKOJIFA | 📅 2026-07-05 | 👁 5,068 | 📂 Claude AI 주식 분석

### 📋 핵심 요약
GitHub 9만 스타 오픈소스 'TradingAgents' 소개. 단일 LLM이 아니라 퀀트/수급/역발상 등 서로 다른 전문성을 가진 다중 AI 에이전트가 실시간 데이터를 놓고 "토론"해서 최종 매수/매도를 결정하는 구조 (2026-06 v0.3.0). AlphaForge로 반도체 섹터 필터링 → 거래대금 상위 20개 추출 → 실시간 뉴스 연동 → 다중 에이전트 분석 → 최종 종목(SK하이닉스, PSK홀딩스 등) 도출까지 시연.

### 🔧 주요 기술/도구
- TradingAgents (오픈소스, github.com/TauricResearch/TradingAgents)
- 다중 AI 에이전트 토론 구조 — 각기 다른 투자 철학(퀀트/수급/역발상/성장)을 가진 에이전트가 상호 반박·합의
- 실시간 뉴스(24~48시간) 데이터 연동
- LLM 모델 교체 가능 (DeepSeek, GPT 등으로 분석 품질 조정)

### 💡 핵심 학습 내용
- "여러 관점의 AI 에이전트가 서로 토론 후 합의"라는 구조는 우리가 안티/코니/만복 3AI 협업에서 이미 하고 있는 방식과 철학적으로 동일 — 이걸 주식 분석에 그대로 적용한 사례
- 오픈소스라 코드를 직접 뜯어볼 수 있음 (9만 스타 = 검증된 구조라는 신호)
- LLM 모델을 상황에 따라 바꿔 끼우는 방식은 며칠 전 확인한 "modelOverrides로 Haiku/Sonnet/Opus 분리" 영상(f7rbKnSRmqM, 6/26 로드맵에 이미 포함)과 같은 맥락

### 🎯 바로보기님 적용 포인트
목표: n8n + AI로 주식 계좌 자동화 (뿌리 ⑤자동화 / ⑦재테크)
- 지금 종가베팅 시스템에 "AI 에이전트 다중 토론" 구조를 얹으면, 단일 AI 판단보다 훨씬 신뢰도 높은 최종 결정을 만들 수 있음 — 다만 이건 구현 난이도가 있어서 3단계(KIS 실계좌) 이후, 4단계(AlphaForge 완성) 단계에서 참고할 항목으로 분류하는 게 맞아 보임
- 당장은 GitHub 코드 구조만 훑어보고 "에이전트 역할 분담 아이디어"만 가져오는 정도로 가볍게 참고 추천
