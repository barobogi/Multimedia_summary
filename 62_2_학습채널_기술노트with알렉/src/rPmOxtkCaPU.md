## 핵심 요약
호주에 거주하는 풀스택 개발자가 바이브 코딩(AI 개발)을 적극 활용하여 개발한 'Sotto Pro(Sotto Voice)' 데스크톱 애플리케이션의 구현 사례와 내부 작동 원리를 소개합니다. 클라우드 비용이나 API 유출 걱정 없이 개인 컴퓨터 자원만을 활용하여 Whisper(음성인식) 및 Qwen 3.5(로컬 번역 LLM) 모델을 100% 로컬 환경에서 구동시키고, 핫키(단축키) 제어를 통해 활성 인스턴스 창에 번역/딕테이션 결과를 자동 타이핑(Type-in)해 주는 온디바이스 음성 생산성 도구의 구축 경험을 배웁니다.

## 주요 기술/도구
- **Local Whisper Model**: 오프라인 상태에서 사용자의 목소리를 듣고 마이크 인풋 스트림을 텍스트로 실시간 변환(Speech-to-Text)해 주는 경량 인공지능 자막 기술.
- **Qwen 3.5 (로컬 LLM)**: 100% 온디바이스 환경에서 기동하며 번역 및 오디오 파일의 텍스트 요약 품질을 엔터프라이즈 급으로 보장해 주는 오픈소스 경량 특화 LLM.
- **Global Hotkey & Simulation**: 특정 키(핫키)를 누르는 동안 음성 녹음을 실행하고, 키를 떼는 순간 활성화된 윈도우 인풋(Cursor AI, 터미널 등)에 텍스트 인풋 시뮬레이션 및 Enter 키 입력 이벤트를 직접 발생시키는 OS 레벨 훅 기술.

## 핵심 학습 내용
